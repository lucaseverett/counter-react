# Counter (React)

This is a basic implementation of a counter using [React](https://reactjs.org/).

## Live Demo

https://code.lucaseverett.dev/counter-react/

## Getting Started

Clone this repo:

```sh
git clone https://github.com/lucaseverett/counter-react.git
```

Install NPM dependencies:

```sh
npm install
```

Start dev server:

```sh
npm start
```
